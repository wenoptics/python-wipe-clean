# Wipe Clean

Wipe clean your console terminal! 

```bash
pip install wipe-clean
```

`wipe-clean` requires Python 3.6.1 and above. Note that Python 3.6.0 is
 not supported due to lack of `NamedTuples` typing.


## Usage

```bash
wipe-clean
```
